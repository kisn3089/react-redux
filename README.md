### React에서 Redux와 Redux-toolkit을 이용하여 간단한 프로젝트를 만들어봤습니다.
```
모든 상태 관리는 Redux Store로 관리 ( useState X )
- 로그인일때 nav요소, Logout 버튼 생성
- 비로그인일때 nav요소 가려지고, form에 ( Email, Password ) Login 버튼 생성
- 숫자 카운트 +, -
- 토글 형식으로 카운트와 +, - 버튼을 숨기고, 보여주기
```

### Technology
- Redux
    - useDispatch
    - useSelector
    - Provider ( index.js )
- Redux-toolkit
    - creatSlice
    - configureStore
